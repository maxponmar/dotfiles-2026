# Tmux Vim-like Keybindings Cheat Sheet

## Pane Navigation

| Key | Action |
|-----|--------|
| `prefix + h` | Move to left pane |
| `prefix + j` | Move to pane below |
| `prefix + k` | Move to pane above |
| `prefix + l` | Move to right pane |

## Window Navigation

| Key | Action |
|-----|--------|
| `prefix + 0-9` | Jump to window 0-9 |
| `prefix + p` | Previous window |
| `prefix + n` | Next window |

## Creating & Closing

| Key | Action |
|-----|--------|
| `prefix + t` | New window |
| `prefix + v` | Split pane vertically |
| `prefix + s` | Split pane horizontally |
| `prefix + Q` | Close current window |

## Pane Resizing

| Key | Action |
|-----|--------|
| `prefix + H` | Resize left (repeat) |
| `prefix + J` | Resize down (repeat) |
| `prefix + K` | Resize up (repeat) |
| `prefix + L` | Resize right (repeat) |

## Copy Mode (Vim-style)

| Key | Action |
|-----|--------|
| `prefix + [` | Enter copy mode |
| `h/j/k/l` | Navigate |
| `v` | Start selection |
| `y` | Copy and exit |
| `prefix + ]` | Paste |

## Utility

| Key | Action |
|-----|--------|
| `prefix + r` | Reload config |

---
*Default prefix: `Ctrl+b`*
